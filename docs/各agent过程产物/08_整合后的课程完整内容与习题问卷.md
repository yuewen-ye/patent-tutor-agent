<h3>[A] 一、法律问题</h3>
<p>被诉方案是否落入专利权保护范围（相同侵权 / 等同侵权）？原告起诉是否超过侵权诉讼时效？</p>
<h3>[A] 二、适用规则</h3>
<ol>
<li><strong>保护范围</strong>：&#34;发明或者实用新型专利权的保护范围以其权利要求的内容为准，说明书及附图可以用于解释权利要求的内容。&#34;（《专利法》第64条第1款）<em>[模拟库已注入]</em></li>
<li><strong>全面覆盖 → 相同侵权</strong>：被诉技术方案具备权利要求<strong>全部</strong>技术特征、逐项对应 → 相同侵权（字面侵权）。<em>[模拟库已注入司法解释要点]</em></li>
<li><strong>等同 → 等同侵权</strong>：被诉技术方案以<strong>基本相同的手段</strong>，实现<strong>基本相同的功能</strong>，达到<strong>基本相同的效果</strong>，且本领域普通技术人员在被诉侵权行为发生时<strong>无需创造性劳动即可联想到</strong> → 等同侵权。<em>[模拟库未提供直接司法解释原文，按通说]</em></li>
<li><strong>诉讼时效</strong>：&#34;侵犯专利权的诉讼时效为三年，自……知道或者应当知道权利受到损害以及义务人之日起计算。&#34;（《专利法》第74条第1款）；《民法典》第188条第1款同为三年、起算一致。<em>[模拟库已注入]</em></li>
</ol>
<h3>[A+B融合] 三、步骤（含对比）</h3>
<ul>
<li><strong>步骤一（定范围）</strong>：以权利要求为准，拆解必要技术特征。</li>
<li><strong>步骤二（相同侵权 / 全面覆盖）</strong>：全部特征一一对应 → 相同侵权。</li>
<li><strong>步骤三（等同侵权）</strong>：字面缺特征时，<strong>逐技术特征</strong>判断&#34;三基本 + 易联想&#34;。注意：等同判断针对<strong>技术特征</strong>而非方案整体；且<strong>禁止反悔原则</strong>可能限制等同范围。</li>
<li><strong>步骤四（时效）</strong>：起算&#34;知道或应知受损 + 义务人&#34;，期间<strong>三年</strong>（与民法典一致）；旧法二年已废止。</li>
</ul>
<h3>[A+B融合] 双列对比表（重点辨析 混淆对 ① / ②）</h3>
<p>| 维度 | 全面覆盖 → 相同侵权 | 等同 → 等同侵权 |
|------|--------------------|----------------|
| 比对方式 | 特征<strong>字面</strong>对应 | 特征<strong>实质等价</strong> |
| 判断标准 | 全部特征都有 | 三基本 + 无需创造性劳动即联想到 |
| 判断对象 | 权利要求全部特征 | <strong>单个技术特征</strong> |
| 结论 | 字面侵权成立 | 等同侵权成立 |</p>
<blockquote>
<p><strong>[B] 口诀</strong>：『全有全算相同，差不多算等同；三个基本加易想，特征比对别整体』。</p>
</blockquote>
<h3>[A] 四、结论</h3>
<p>具备全部特征 → 相同侵权；字面缺特征但构成&#34;三基本 + 易联想&#34; → 等同侵权；起诉未超三年时效则获支持。</p>
<blockquote>
<p>来源统计：专家A段落 5 处 / 专家B段落 3 处 / A+B融合 2 处。</p>
</blockquote>

<h2>二、本节习题问卷（诊断 + 巩固 · 供反馈阶段读取）</h2>
<blockquote>
<p>覆盖：相同/等同侵权判断、等同判断对象、诉讼时效（跨法衔接）。附答案与知识点映射。</p>
</blockquote>
<p><strong>Q1（单选）</strong> 被诉产品缺少权利要求中的技术特征 X，但用 Y 替换；Y 与 X 手段、功能、效果基本相同，且本领域技术人员易联想到。该情形构成：<br>
A. 相同侵权　B. 等同侵权　C. 不侵权　D. 视情况而定<br>
<strong>答案</strong>：B　<strong>知识点</strong>：等同原则与等同侵权　<strong>解析</strong>：字面缺特征但构成&#34;三基本 + 易联想&#34; → 等同侵权。</p>
<p><strong>Q2（单选）</strong> 根据现行《专利法》，侵犯专利权的诉讼时效期间为：<br>
A. 2 年　B. 3 年　C. 5 年　D. 20 年<br>
<strong>答案</strong>：B　<strong>知识点</strong>：侵权诉讼时效 / 民法典诉讼时效　<strong>解析</strong>：第74条与民法典第188条均为 3 年；旧法 2 年已废止。</p>
<p><strong>Q3（判断）</strong> 等同侵权判断应将被诉技术方案与权利要求整体进行比对。 （　）<br>
<strong>答案</strong>：×（错误）　<strong>知识点</strong>：等同原则与等同侵权　<strong>解析</strong>：等同判断针对<strong>单个技术特征</strong>，而非方案整体。</p>
<p><strong>Q4（单选）</strong> 侵权诉讼时效的起算点是：<br>
A. 侵权行为发生之日　B. 权利人知道或应当知道权利受损及义务人之日<br>
C. 专利权授权之日　D. 起诉之日<br>
<strong>答案</strong>：B　<strong>知识点</strong>：侵权诉讼时效　<strong>解析</strong>：起算&#34;知道或应知受损 + 义务人&#34;。</p>
<p><strong>Q5（简答）</strong> 用一句话区分&#34;全面覆盖原则&#34;与&#34;等同原则&#34;。<br>
<strong>参考答案</strong>：全面覆盖看特征<strong>字面对应</strong>（全有 = 相同侵权），等同看特征<strong>实质等价</strong>（三基本 + 易联想 = 等同侵权）。　<strong>知识点</strong>：全面覆盖原则 / 等同原则</p>
<p><strong>Q6（案例）</strong> 学员 L 的幕墙专利权利要求含&#34;卡扣 A&#34;；竞品用&#34;磁吸快装件&#34;实现相同连接功能与效果，行内人易联想到。请判断是否侵权，并说明属何种侵权及理由。<br>
<strong>参考答案</strong>：构成侵权，属<strong>等同侵权</strong>。理由：竞品字面缺少&#34;卡扣 A&#34;，但以&#34;磁吸快装件&#34;这一基本相同的手段实现基本相同的功能与效果，且本领域技术人员无需创造性劳动即可联想到，符合等同要件（逐技术特征比对）。　<strong>知识点</strong>：等同原则与等同侵权</p>
<blockquote>
<p>习题来源归属：Q1/Q3/Q5/Q6 为 A+B融合考点；Q2/Q4 为跨法衔接（专家A法条骨架 + 专家B口诀提示）。</p>
</blockquote>
