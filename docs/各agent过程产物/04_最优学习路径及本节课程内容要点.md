<h2>二、本节课程内容要点（当前节点：等同原则与等同侵权）</h2>
<blockquote>
<p>面向学员 L（掌握概率 0.22，具体感知/视觉/顺序型，困惑状态）。深度 = 深入，重点 = 辨析。</p>

<h3>核心知识点</h3>
<ol>
<li><strong>权利要求保护范围</strong>：以权利要求内容为准，说明书及附图用于解释（第64条）。</li>
<li><strong>全面覆盖原则 → 相同侵权</strong>：被诉方案具备权利要求<strong>全部</strong>技术特征（逐一对应）→ 字面侵权。</li>
<li><strong>等同原则 → 等同侵权</strong>：字面缺特征，但该特征被&#34;以基本相同的手段，实现基本相同的功能，达到基本相同的效果，且本领域普通技术人员在侵权行为发生时<strong>无需创造性劳动即可联想到</strong>&#34;的特征所替换 → 等同侵权。</li>
<li><strong>判断对象</strong>：等同判断针对<strong>单个技术特征</strong>，不是方案整体。</li>
<li><strong>禁止反悔原则</strong>（提示）：专利权人为获授权所作限缩性修改或陈述，会限制其主张等同的范围。</li>
</ol>
<h3>必须对比的易混淆点（混淆对 ① / ②）</h3>
<ul>
<li>全面覆盖 = 看&#34;字面&#34;（特征字面对应）；等同 = 看&#34;实质等价&#34;（三基本 + 易联想）。</li>
<li>相同侵权与等同侵权都导致侵权成立，但<strong>依据与判断标准不同</strong>。</li>
</ul>
<h3>跨法衔接提示（混淆对 ③）</h3>
<ul>
<li>侵权诉讼时效：现行《专利法》第74条 = <strong>3 年</strong>，起算&#34;知道或应知受损 + 义务人&#34;；与《民法典》第188条一致。旧法&#34;2 年&#34;已废止。（详细放在节点 4–5）</li>
</ul>
<h3>应试关键词</h3>
<ul>
<li>题干出现&#34;特征一一对应 / 完全相同&#34; → 全面覆盖 / 相同侵权。</li>
<li>题干出现&#34;手段近似 / 功能效果基本相同 / 容易想到&#34; → 等同 / 等同侵权。</li>
<li>题干出现&#34;缺少某技术特征但……&#34; → 进入等同判断，逐特征比对。</li>
</ul>
<h3>本节评估</h3>
<ul>
<li>案例题：权利要求有特征 X；被诉产品无 X，但以 Y 实现基本相同功能效果且易联想到 → 判断是否侵权、属何种侵权。</li>
</ul>
