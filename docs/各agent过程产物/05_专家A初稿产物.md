<h2>教学正文（问题—规则—适用—结论 四步法 · 人读版）</h2>
<h3>一、法律问题</h3>
<p>争点：被诉侵权技术方案是否落入原告专利权保护范围？若落入，属于<strong>相同侵权</strong>还是<strong>等同侵权</strong>？以及，原告起诉是否超过侵权诉讼时效？</p>
<h3>二、适用规则</h3>
<ol>
<li><strong>《专利法》第六十四条第一款</strong>：&#34;发明或者实用新型专利权的保护范围以其权利要求的内容为准，说明书及附图可以用于解释权利要求的内容。&#34; <em>[模拟库已注入对应法条片段]</em></li>
<li><strong>全面覆盖原则</strong>（司法解释通说）：被诉技术方案具备权利要求<strong>全部</strong>技术特征、逐项对应 → <strong>相同侵权</strong>（字面侵权）。 <em>[模拟库已注入司法解释要点]</em></li>
<li><strong>等同原则</strong>（司法解释通说）：被诉技术方案以<strong>基本相同的手段</strong>，实现<strong>基本相同的功能</strong>，达到<strong>基本相同的效果</strong>，且本领域普通技术人员在被诉侵权行为发生时<strong>无需创造性劳动即可联想到</strong> → 该差异特征为等同特征 → <strong>等同侵权</strong>。 <em>[模拟库未提供直接司法解释原文，以下按通说表述]</em></li>
<li><strong>《专利法》第七十四条第一款</strong>：&#34;侵犯专利权的诉讼时效为三年，自专利权人或者利害关系人知道或者应当知道权利受到损害以及义务人之日起计算。&#34; <em>[模拟库已注入对应法条片段]</em></li>
<li><strong>《民法典》第一百八十八条第一款</strong>：&#34;向人民法院请求保护民事权利的诉讼时效期间为三年……诉讼时效期间自权利人知道或者应当知道权利受到损害以及义务人之日起计算。&#34; <em>[模拟库已注入对应法条片段]</em></li>
</ol>
<h3>三、规则适用</h3>
<ul>
<li><strong>步骤一（定范围）</strong>：以权利要求为准，拆解必要技术特征。</li>
<li><strong>步骤二（相同侵权 / 全面覆盖）</strong>：将被诉方案与权利要求逐项比对；全部特征一一对应 → 相同侵权。</li>
<li><strong>步骤三（等同侵权）</strong>：若字面不全部对应，逐技术特征判断是否&#34;三基本 + 无需创造性劳动即可联想到&#34;。注意：等同判断对象是<strong>技术特征</strong>而非方案整体；且禁止反悔原则可能限制等同范围。 <em>[模拟库未提供直接法条依据]</em></li>
<li><strong>步骤四（诉讼时效）</strong>：起算点为&#34;知道或应知受损 + 义务人&#34;，期间<strong>三年</strong>（与民法典一致）；旧法二年规定已废止。</li>
</ul>
<h3>四、结论</h3>
<p>具备全部特征 → 相同侵权；字面缺特征但构成&#34;三基本 + 易联想&#34; → 等同侵权；起诉未超三年时效则获支持。</p>
<blockquote>
<p><strong>常见误区 / 易混淆点</strong>：① 全面覆盖原则与等同原则不可混为一谈；② 等同侵权须逐技术特征判断，不能整体比对；③ 侵权诉讼时效现行三年，旧法二年已废止，非特殊短期。</p>
</blockquote>
