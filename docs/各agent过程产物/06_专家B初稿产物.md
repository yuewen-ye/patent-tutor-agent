<h2>教学正文（生动版 · 人读版）</h2>
<h3>1. 场景导入</h3>
<p>你是建筑师，给&#34;幕墙单元板块&#34;申请了专利，权利要求写：<em>板块含卡扣 A + 密封条 B + 排水槽 C</em>。竞争对手做出的产品没用卡扣 A，而是用一种&#34;磁吸快装件&#34;把板块拼上——功能、效果跟你的一模一样，行内人一看就想得通。这算侵权吗？</p>
<h3>2. 人话解释</h3>
<p>专利侵权就两问：
① 你的东西是不是&#34;全抄&#34;了权利要求（全部特征都有）→ <strong>相同侵权</strong>；
② 不是全抄，但换了个&#34;差不多的手段&#34;达到一样效果，行内人轻易能想到 → <strong>等同侵权</strong>。</p>
<h3>3. 法条回扣</h3>
<ul>
<li>保护范围：&#34;以权利要求的内容为准&#34;（《专利法》第64条第1款）。 <em>[模拟库已注入]</em></li>
<li>相同侵权 = 全面覆盖：权利要求特征一个不落全中。</li>
<li>等同侵权：&#34;基本相同的手段 + 功能 + 效果，且无需创造性劳动即联想到&#34;（《专利法》相关司法解释通说）。 <em>[模拟库未提供直接依据]</em></li>
<li>时效：侵权诉讼时效&#34;三年&#34;，起算&#34;知道或应知受损 + 义务人&#34;（《专利法》第74条；《民法典》第188条）。 <em>[模拟库已注入]</em></li>
</ul>
<h3>4. 类比 / 口诀</h3>
<p><strong>口诀</strong>：『全有全算相同，差不多算等同；三个基本加易想，特征比对别整体』。</p>
<p><strong>类比边界</strong>：&#34;等同&#34;不是&#34;随便差不多&#34;——必须手段/功能/效果三基本都沾边，且行内人不用费劲就能想到。像&#34;把螺丝换成卡扣&#34;这种显而易见替换，才是等同；若是换了一套原理完全不同的结构，通常不构成等同。</p>
<h3>5. 应试提示</h3>
<ul>
<li>题干&#34;特征一一对应 / 完全相同&#34; → 全面覆盖 / 相同侵权。</li>
<li>题干&#34;手段近似 / 功能效果基本相同 / 容易想到&#34; → 等同 / 等同侵权。</li>
<li>题干&#34;缺少某技术特征但……&#34; → 进入等同判断，<strong>逐特征</strong>比对。</li>
<li>时效题：现行<strong>三年</strong>（旧法二年已作废），起算&#34;知道或应知受损 + 义务人&#34;。</li>
</ul>
